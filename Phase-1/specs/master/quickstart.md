# Quickstart: Todo Console Application

## Prerequisites
- Python 3.13 or higher
- No additional dependencies required

## Running the Application

1. Navigate to the project directory
2. Run the application:
   ```bash
   python src/cli/main.py
   ```

## Using the Application

1. The application will display a menu with available options
2. Enter the number corresponding to the action you want to perform
3. Follow the prompts to provide required information

## Available Features

- **Add Todo**: Create a new todo with title and optional description
- **View Todos**: List all todos with their status
- **Update Todo**: Modify title or description of an existing todo
- **Delete Todo**: Remove a todo by ID
- **Mark Complete/Incomplete**: Update the completion status of a todo
- **Exit**: Quit the application

## Example Usage

1. Start the application
2. Choose option 1 to add a new todo
3. Enter a title (e.g., "Buy groceries")
4. Optionally enter a description (or press Enter to skip)
5. The todo will be created with a unique ID
6. Choose option 2 to view all todos
